# Shimarin

## SystemDependencies

- firebase-cli
- yarn
- direnv allow
- Nodejs

## Development

1. Run `yarn && yarn setup`
2. Run `direnv allow`
3. Run `firebase use --add`
4. Run `firebase serve --only functions & yarn start`

## Deployment

1. Run `yarn run build`
2. Run `firebase use --add`
3. Run `firebase deploy`
